# Arib Digital Studio - Professional Portfolio

This is a professional portfolio website built with modern web technologies, showcasing digital marketing solutions, skills, and projects.

## Tech Stack

- **Framework**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Database & Auth**: [Firebase Realtime Database](https://firebase.google.com/docs/database)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## Key Features

- **Dynamic Content**: All content is managed through a Firebase Realtime Database.
- **Admin Panel**: Secure dashboard for managing profile, skills, services, projects, and testimonials.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Service Reviews**: Client feedback system.
- **Lead Capture**: Contact forms and service request tracking.
- **Analytics**: Built-in visitor tracking and page view analytics.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Set up your Firebase configuration in `src/lib/firebase.ts`.
4. Run the development server:
   ```sh
   npm run dev
   ```

## Deployment

The project can be deployed easily on platforms like Vercel or Netlify.
